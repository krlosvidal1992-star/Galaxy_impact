# 🚀 Galaxy Impact :

¡Bienvenido a mi proyecto! Esta es una reinterpretación del clásico juego **Space Impact**, desarrollada como parte de mi aprendizaje en **Penguin Academy**. En esta versión, combinamos la nostalgia de los juegos de naves.

## 🎮 Características del Juego
*   **Menú Principal Moderno**: Interfaz fluida con botones personalizados.
*   **Sistema de Niveles**: Dificultad progresiva a medida que eliminas enemigos.
*   **Jefe Final**: Un enfrentamiento desafiante al alcanzar el nivel 5.
*   **Gráficos Personalizados**: Uso de iconos y logos adaptados para dispositivos móviles.

## 🛠️ Tecnologías Utilizadas
*   **Flutter & Dart**: Framework principal para el desarrollo multiplataforma.
*   **Flame Engine**: Motor de juegos 2D de alto rendimiento para Flutter.
*   **Git & GitHub**: Control de versiones y despliegue del código.

## 📱 Cómo probar la aplicación
Si deseas instalar el juego en tu dispositivo Android (como un Xiaomi), sigue estos pasos:

1.  Descarga el archivo `app-release.apk` ubicado en la carpeta de lanzamientos.
2.  Transfiere el archivo a tu teléfono.
3.  Habilita la instalación desde "Fuentes desconocidas" en los ajustes de seguridad.
4.  ¡Instala y disfruta!
